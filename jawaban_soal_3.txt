Fork kurang lebih sama dengan klon, namun klon dilakukan untuk dikerjakan di lokal kita. sedangkan fork dilakukan untuk dikerjakan dalam repository baru di akun github kita. untuk selanjutnya kita kembangkan namun tetap ada alamat asal dari fork yang kita lakukan sebagai bentuk kredit.
Pull request adalah kita meminta pada master (pemilik repository) untuk menerapkan perubahan yang kita buat (branch milik kita). artinya merge antara master dan branch kita. pull request juga dilakukan setelah kita fork repository orang lain dan ingin menyarankan perubahan pada repository awal. pull request hanya dapat diterima oleh pemilik repository, umumnya untuk melakukan merge dengan master.
